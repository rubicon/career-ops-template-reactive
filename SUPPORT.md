# Support

## Questions and discussion

For how-to questions and general discussion, use
[GitHub Discussions](https://github.com/rubicon/career-ops-template-reactive/discussions).
Please do not open an issue for a question.

## Bugs and feature requests

Use the issue templates:

- [Bug report](https://github.com/rubicon/career-ops-template-reactive/issues/new?template=bug_report.yml)
- [Feature request](https://github.com/rubicon/career-ops-template-reactive/issues/new?template=feature_request.yml)

## Security

Do not report security issues in public. See [SECURITY.md](SECURITY.md).

## career-ops itself

Questions about career-ops (the host project) belong in the
[career-ops repository](https://github.com/santifer/career-ops), not here.
